What is JDK?
- JDK or java development kit is everything we need to have to develop a java program.
It consists of JRE(Java Runtime Environment), JVM (Java Virtual Machine), Compiler, debugger etc.

What is JRE?
- JRE is inside JDK as a part of JDK. It stands for Java runtime environment and provides a runtime 
environment to run java files. It contains all java core libraries, security files etc.

What is JVM?
- JVM stands for Java Virtual Machine. This is inside of or a component of JRE. This converts the bytecode
into actual machine code that the machine can understand.

What is bytecode?
- Bytecode is an intermediate level code. When a .java class is compiled, it is converted to an intermediate byte code file .class
It is a platform independent code and can run on any system. JVM then converts this bytecode into machine code and runs it.

What does “write once, run anywhere” mean :
- This means that java compiled code can be run anywhere. It can be written once and then executed on any system.
This is because once a java program is compiled , it is converted to byte code(.class file). 
This byte code is platform independent , independent of the OS it is run on, hence any system with
a JVM can run it.
